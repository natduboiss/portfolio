Parallax Web Scroll

Goal

Create a webpage and animate this page to mimic a parallax scrolling feature of a web page. It should use multiple layers which seem to move at different speeds to create a sensation of depth and an interesting browsing experience.

Insight

The examples that I saw online about parallax web scrolling were all above ground scrolling into the ground. I wanted to do it a bit differently and use the sea. As you scroll further down the page you would go deeper in the depths of the water, discovering new things.
